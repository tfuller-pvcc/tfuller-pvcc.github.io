All About Github

Q. What is a version control system?

A. Class of systems responsible for managing changes to computer programs

Q. What is a Git?

A. A version control system used for tracking changes

Q. What is Github?

A. An internet hosting service for software development and version control

Q. What is the difference?

A. Github lets you manage respositories

Q. Who started Github and how was it started?

A. Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, and Scott Chacon, with Ruby on Rails

Q. What company owns it now?

A. Microsoft

Q. How much does a Github cost?

A. Free

Q. What is the Octocat

A. Mascot of Github

Git/Github Terms:

  • repository: a central location where data is stored
  • commit: A change to a file
  • fork: A copy of a repository that you manage
  • push: Uploading a local repository to a remote repository
  • pull requests: Communicating changes you've made to a repository
  • workflows: A workflow is a configurable automated process that will run one or more jobs
  • issues: Lets you track your work on Github where development happens
  • raw button: Opens up a file in a raw form
  • blame button: Used to examine the contents of a file and who made changes
